# Trippy is a travel agency website made with react (SPA)
<small>It is a side project </small>

# Pages
- Home
- Services
- About Us
- Contact Us
- Sign Up 

# Features
- React Router
- Re-useable components
- FontAwesome icons

# How to start
- clone repo
- open in terminal
- execute <code>npm install</code>
- then run react with <code>npm start</code>
- then enjoy the app
